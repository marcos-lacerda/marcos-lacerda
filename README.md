## 🧭 About Me

I build digital products that work — from the first commit to the first real transaction.

Over the past **10+ years**, my career has followed one thread: using technology to solve real business problems. I'm not the dev chasing the prettiest code on the wall — I'm the one asking *"what problem are we solving?"* before opening the editor.

At **Turbi**, I joined on day zero and was the sole iOS developer for the first 3 years — building the only channel of a fully digital car rental operation. The app handled facial validation (FaceTec 3D Liveness), remote vehicle control via Bluetooth, IoT telemetry, PCI-compliant payments, and more. Later I moved to backend (Node.js) and Flutter to solve operational pain points, and closed with a real-time IoT telemetry pipeline using GCP PubSub and MongoDB Time Series.

Now I'm building **[Agent Trip](https://github.com/agent-trip)** — a travel planning app I designed and develop end-to-end as a solo founder. Flutter + NestJS + GCP + Firebase. 200+ downloads and first real transactions in 3 months on the stores.
